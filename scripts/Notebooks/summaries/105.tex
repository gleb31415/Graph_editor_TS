\subsubsection*{Условие баланса моста}

\[
\begin{cases}
I_1 + I_3 - I_5 = 0 \\
I_2 + I_4 + I_5 = 0 \\
I_1R_1 + I_5R_5 - I_2R_2 = 0 \\
I_3R_3 + I_5R_5 - I_4R_4 = 0
\end{cases}
\]

При $I_5=0$:
\[
R_2R_3 = R_1R_4.
\]

Эквивалентное сопротивление:
\[
R_{\rm экв} = \frac{(R_1 + R_3)(R_2 + R_4)}{R_1 + R_2 + R_3 + R_4}.
\]
